# Usage
Edit pom.xml as necessary for correct heroku deployment.
Create .env to
```
MONGODB_USER=dbuser
MONGODB_PASS=dbpassword
MONGODB_NAME=dbname
MONGODB_HOST=dbhosturl
MONGODB_PORT=dbport
```
